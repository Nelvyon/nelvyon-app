# NELVYON V2-A1 — Demo Script (8-10 min)

Goal: prove Revenue Operations Core is usable, sellable, and operationally honest.

## 1) Entry point (30s)

- Open `CRM > Deals`.
- Confirm workspace context is selected.

Talk track:
"This is the new commercial execution layer on top of NELVYON, scoped per workspace."

## 2) Pipeline depth + filters (1-2 min)

- Show deals list with stage and owner filters.
- Filter by one stage and one owner.
- Reset filters and show full list again.

Talk track:
"Teams can focus pipeline slices quickly without exporting data."

## 3) Conversion/leakage + risk visibility (1-2 min)

- Show mini conversion/leakage panel.
- Show Deals at risk panel.
- Open one at-risk deal from the list.

Talk track:
"The objective is not vanity analytics; it is fast visibility on leakage and risk."

## 4) Owner accountability + next step (2 min)

- In deal detail, edit:
  - stage
  - owner
  - next step
- Save changes and confirm persisted state.

Talk track:
"Every deal has clear ownership and an explicit next action."

## 5) Follow-up activity (1-2 min)

- Create a follow-up activity with due date.
- Show it appears in the deal follow-up list.

Talk track:
"Follow-up is captured in-product, tied to deal context, without ad-hoc side channels."

## 6) Role and isolation check (1 min)

- Show behavior with a non-operator user (read-only / blocked mutation).
- Confirm no cross-workspace leakage by switching workspace context.

Talk track:
"Commercial execution stays controlled: RBAC and tenant scope are enforced."

## 7) Close (30s)

- Summary sentence:
"V2-A1 turns CRM data into accountable pipeline execution with risk visibility and practical follow-up."
