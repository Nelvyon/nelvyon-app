# NELVYON v1.1 — Demo Script (10-15 min)

Audience: enterprise prospect, operator lead, or investor.
Goal: prove NELVYON is a serious self-service product without daily human intervention.

## 0) Setup (30-60s)

- Confirm you are in a demo workspace.
- Use two users if possible:
  - Admin user (can run checkout).
  - Operator/member user (cannot change plan).

## 1) Login + workspace context (1 min)

- Open sign-in and authenticate.
- Show workspace selector and explain tenant isolation:
  - All operational and billing actions are scoped to selected workspace.

Talk track:
"NELVYON works workspace-by-workspace. Access and data are isolated by context, not shared globally."

## 2) Roles and permissions (1-2 min)

- Open Settings.
- Show member/invite/profile actions and role-based behavior.
- If using two users, show admin can do actions that member/operator cannot.

Talk track:
"Critical actions are role-gated. We avoid ambiguous privilege behavior in the UI."

## 3) Onboarding self-activation (1-2 min)

- Open home/checklist.
- Complete or review activation steps.
- Show that a new workspace can move forward without support intervention.

Talk track:
"A new workspace can activate independently with guided milestones."

## 4) Help center + structured forms (2 min)

- Open Help center.
- Search a practical question.
- Open structured form (bug/help/feedback) and show actionable path.

Talk track:
"Support is productized: users can self-serve answers and escalate with structured intake when needed."

## 5) Bot first-line support (1-2 min)

- Open bot panel.
- Ask one known question (returns useful action).
- Ask one uncertain question (shows honest handoff path).

Talk track:
"Bot behavior is intentionally honest: useful when confident, handoff when uncertain."

## 6) Operations confidence (OS) (1-2 min)

- Open OS overview.
- Show health banner, recent jobs, and operational playbook.
- Highlight practical actions (retries, queue awareness, webhook checks).

Talk track:
"Operations signals are actionable. Teams see what needs attention now."

## 7) Billing + upgrade self-service journey (3 min)

- Open Billing overview:
  - plan, usage, invoices, risk signals.
- Open Upgrade flow:
  - select plan + cycle
  - start checkout
  - return and verify payment
  - confirm active subscription + refreshed billing state

Talk track:
"Commercial journey is end-to-end: choose, checkout, verify, reflect in billing—without manual support."

## 8) Audit & security visibility (1-2 min)

- Open Settings > Audit & security.
- Show event type, actor, timestamp, status/severity + filters.
- Return to Settings and show recent security activity snippet.

Talk track:
"Enterprise trust starts with visible traceability: who did what, when, and with what result."

## 9) Close (30s)

- Summarize in one sentence:
"NELVYON v1.1 delivers self-activation, first-line support, operational visibility, secure role gating, and real self-service billing upgrades."
