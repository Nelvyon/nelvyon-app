## NAME

Formación y capacitación digital Premium OS Standards for NELVYON v1 (client delivery checklist — **paperwork layer only**)

## Purpose

This **Formación / capacitación Premium** template is **OS-facing delivery paperwork**: statuses, evidence, **type badges** (taller_presencial, curso_online, mentoria, webinar, manual_tecnico, onboarding_herramienta, programa_continuo), and links — it does **not** provision LMS tenants, publish SCORM packages, or call external training APIs from this surface.

Suggested touchpoints for honest verification:

- `/os/advisor-empresarial-premium/preview` — when executive coaching narratives overlap training scope paperwork  
- `/help` — accessibility, IP, or HR-sensitive training escalations  

---

## Scope (minimum real)

- **Delivery discipline:** Audience, outcomes, prerequisites, and success metrics documented before client sign-off.
- **Honesty:** Modality mix (live, async, hybrid) and facilitator coverage explicit — no “unlimited live hours” from OS alone.
- **Assessment:** Evaluation method and data handling described — template does not run quizzes or grade learners.

---

## Limits (explicit non-goals)

- No LMS, webinar platform, or credentialing API connections from this OS template.
- No creation of real courses, SCORM, or video lesson files from this preview.
- Does not modify closed product fronts; does not open `/crm/deals`.

## Premium checklist (digital training delivery)

| Area | Premium expectation |
| --- | --- |
| READY | Golden path / `pnpm gate` discipline when training-related surfaces ship in the same release train. |
| NEEDS DIAGNOSIS | Skills gaps, roles, and constraints documented. |
| CURRICULUM DESIGN | Outcomes, modules, and timeboxing explicit. |

| DELIVERY / INSTRUCTION | Schedule, cohort size, and language/accessibility requirements explicit. |

| CERTIFICATION | Issuance rules and registry posture explicit — no auto-badge from template. |
| REPORTING & FOLLOW-UP | Completion metrics and limitations explicit — template records expectations only. |
