## 1. Ideal Customer Profile (ICP)

### Perfil primario — **Growth SMB**

| Atributo | Ideal |
|----------|-------|
| **Tipo** | Pyme B2B o B2C con producto/servicio validado |
| **Facturación** | €300K – €5M anual (orientativo) |
| **Equipo marketing** | 0–3 personas (necesita externalizar ejecución) |
| **Madurez digital** | Web existente o lanzamiento inminente |
| **Presupuesto marketing anual** | ≥ €15K (fee NELVYON + medios) |
| **Decisor** | CEO, CMO, Director comercial — accesible en ≤ 2 reuniones |
| **Geografía** | España / UE (ES/EN) · remoto OK |
| **Motivación** | Crecer leads, ventas online o profesionalizar marca |

### Perfil secundario — **Startup / Scale-up**

| Atributo | Ideal |
|----------|-------|
| **Stage** | Post-seed o Serie A con runway ≥ 12 meses |
| **Necesidad** | Launch bundle (marca + web + ads) en plazo corto |
| **Ticket** | €8K – €25K proyecto inicial |
| **Criterio** | Founder pragmático, feedback rápido |

### Perfil terciario — **Ecommerce establecido**

| Atributo | Ideal |
|----------|-------|
| **GMV** | ≥ €200K/año online |
| **Necesidad** | Ecommerce Pro + paid media + automatización |
| **Ticket** | €12K+ anual recurrente posible |

## 3. Anti-ICP (no priorizar)

| Perfil | Por qué |
|--------|---------|
| Presupuesto total < €1.500 sin upsell claro | No cubre QA + margen |
| Solo "presupuesto lo más barato" | Race to bottom |
| Sin decisor en 3+ semanas | Pipeline muerto |
| Expectativa ROI garantizado en contrato | Fuera modelo NELVYON |
| Sector ilegal / reputación tóxica | Riesgo marca |
| Quiere "todo el catálogo" por precio de 1 landing | Scope irreal |

## 4. Red flags

### Críticas — **No firmar sin mitigación escrita**

| # | Señal | Riesgo | Acción |
|---|-------|--------|--------|
| RF-01 | Rechaza pagos por hitos | Impago | 100% upfront o walk away |
| RF-02 | Historial impago proveedores | Cobro | Referencias / prepago |
| RF-03 | Exige IP antes de pago final | Disputa | Cláusula estándar |
| RF-04 | Contenido engañoso / ilegal pedido | Legal | Rechazar |
| RF-05 | > 5 stakeholders sin decisor único | Parálisis | Exigir sponsor |

### Altas — **Proceed with caution**

| # | Señal | Riesgo | Acción |
|---|-------|--------|--------|
| RF-10 | "Queremos lo de la competencia por la mitad" | Margen | Re-tier o no bid |
| RF-11 | Sin accesos (dominio, ads) a día 0 | Retraso | Checklist accesos en contrato |
| RF-12 | Revisiones ilimitadas implícitas | Scope creep | Rondas en SOW |
| RF-13 | Comparan solo por precio, no entregables | Churn | Educar tiers + QA |
| RF-14 | Proyecto urgente < 48h sin premium | Calidad | Premium + rush fee o declinar |

### Medias — **Documentar en CRM**

| # | Señal | Acción |
|---|-------|--------|
| RF-20 | Primera vez externalizando marketing | Más educación en propuesta |
| RF-21 | Industria muy regulada sin compliance interno | Anexo responsabilidad cliente |
| RF-22 | Cambios de prioridad frecuentes en discovery | SOW estricto |

## 5. Lead scoring

**Uso:** Priorizar follow-up · umbral propuesta **≥ 60** · firma objetivo **≥ 75**

### Criterios (máx. 100 puntos)

| Dimensión | Puntos | 0 | 5 | 10 |
|-----------|--------|---|---|-----|
| **A. Fit ICP** | 0–20 | Fuera ICP | Parcial | ICP primario |
| **B. Presupuesto declarado** | 0–20 | < €1.5K | €1.5K–€7K | ≥ €7K |
| **C. Urgencia / timing** | 0–10 | Sin fecha | 3–6 meses | ≤ 8 semanas |
| **D. Decisor accesible** | 0–15 | Desconocido | Influencer | Decisor en call |
| **E. Necesidad clara** | 0–15 | Vaga | 1 servicio claro | Bundle / roadmap |
| **F. Calidad lead** | 0–10 | Spam / frío | Inbound genérico | Referido / inbound cualificado |
| **G. Red flags** | −30 a 0 | RF crítica (−30) | RF alta (−15) | Ninguna (0) |

### Fórmula

```
Score = A + B + C + D + E + F + G
```

### Acciones por rango

| Score | Prioridad | Acción |
|-------|-----------|--------|
| **85–100** | A | Propuesta ≤ 3 días · AM senior |
| **75–84** | B | Propuesta estándar · discovery completo |
| **60–74** | C | Nurture · clarificar presupuesto/decisor |
| **40–59** | D | Email educativo · recontacto 30d |
| **< 40** | E | Archivar o lista fría |

## 6. Preguntas discovery (BANT+)

| Área | Pregunta clave |
|------|----------------|
| **Budget** | ¿Rango inversión fee agencia este trimestre? ¿Medios ads aparte? |
| **Authority** | ¿Quién firma y aprueba entregables? |
| **Need** | ¿Qué problema resuelve esto en 90 días? |
| **Timeline** | ¿Fecha lanzamiento o campaña? |
| **Fit** | ¿Web/tracking actual? ¿Qué probaste antes? |
| **Success** | ¿Métrica que define éxito? (leads, ROAS, tráfico) |
