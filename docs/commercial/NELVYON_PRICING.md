## 1. Principios de precio

| Principio | Regla |
|-----------|-------|
| **Tier-first** | Precio = alcance tier, no horas abiertas |
| **SOP = scope** | Lo no listado en SOP = change order |
| **QA incluido** | Todos los tiers incluyen QA G3 NELVYON |
| **Medios aparte** | Presupuesto ads, stock, licencias fuera de fee setup |
| **Descuentos** | Solo bundles o contrato anual — aprobación Ops |

**Nota legal comercial:** Precios orientativos. Propuesta firmada prevalece. IVA no incluido salvo indicación.

## 6. Condiciones de pago estándar

| Tipo proyecto | Estructura |
|---------------|------------|
| **< €3.000** | 100% firma o 50/50 firma-entrega |
| **€3.000 – €10.000** | 40% firma · 40% hito mitad · 20% entrega |
| **> €10.000** | 30% firma · 40% hito · 30% aceptación |
| **Retainer mensual** | Mes adelantado · permanencia mínima según tier |

**Penalización retraso cliente** (brief, feedback, accesos): SLA pausa — sin penalización NELVYON.

## 7. Exclusiones globales (no incluido en precio)

| Ítem | Notas |
|------|-------|
| Presupuesto publicidad (Meta, Google, TikTok) | Paga cliente directo plataforma |
| Fotografía producto profesional | Presupuesto aparte o cliente provee |
| Copy legal / revisión abogado | Cliente o tercero |
| Traducciones multi-idioma masivas | CR por idioma |
| Integraciones ERP/custom dev | AUTOMATION premium o dev aparte |
| Dominio, hosting, email corporativo | Cliente o gestión aparte |
| Licencias stock premium (Shutterstock, etc.) | Cliente |

## 8. Matriz rápida: ¿qué tier recomendar?

| Señal cliente | Tier |
|---------------|------|
| "Quiero probar si funciona" | Starter |
| "Necesito algo serio para crecer" | **Professional** |
| "Queremos ser referentes en el sector" | Premium |
| Presupuesto < €800 servicio | No encaja — referir o solo Starter logo/landing |
| Quiere todo + ayer | Premium + timeline realista en propuesta |

## 9. Upgrade / downgrade mid-project

| Cambio | Regla precio |
|--------|--------------|
| Starter → Professional | Diferencia prorrateada + CR firmado |
| Añadir servicio | Propuesta adicional o bundle upgrade |
| Reducir scope | Downgrade tier; no reembolso trabajo realizado |
