## 2. Proceso paso a paso

### Fase 0 — Intake (D0–2) · Account

Brief · acceso Ads Manager · landing mobile verificada · OS `TIKTOK-[CLIENTE]` · kick-off 45 min.

### Fase 1 — Research (D1–3) · Freelancer

| Paso | Acción | Evidencia |
|------|--------|-----------|
| 1.1 | TikTok Creative Center: ads competencia sector | Screenshots |
| 1.2 | 5 hooks candidatos alineados oferta | Doc hooks |
| 1.3 | Definir estructura video 15–30s (hook 1s, body, CTA) | Storyboards |
| 1.4 | Opcional: AdsTiktokAgent / AdsPremiumAgent copy borrador | Internal |

### Fase 2 — Estrategia (D3–4) · Freelancer

| Paso | Acción | Evidencia |
|------|--------|-----------|
| 2.1 | Plan campaña: objetivo, audiencia, placement | PDF estrategia |
| 2.2 | Naming convention | Tabla |
| 2.3 | Calendario creative rotation | Calendario |
| 2.4 | **Aprobación cliente** | Email |

### Fase 3 — Producción video (D4–10) · Freelancer / creator

| Paso | Acción | Evidencia |
|------|--------|-----------|
| 3.1 | Grabación o motion según storyboard | RAW |
| 3.2 | Edición 9:16, subtítulos quemados | MP4 |
| 3.3 | 3–8 variantes según tier | Files nombrados |
| 3.4 | **Revisión** creatividades | OK cliente |

### Fase 4 — Tracking (D8–9) · Freelancer

| Paso | Acción |
|------|--------|
| 4.1 | Pixel TikTok + eventos (CompletePayment, SubmitForm, etc.) |
| 4.2 | Test evento con TikTok Pixel Helper |
| 4.3 | UTMs en destino |

### Fase 5 — Setup Ads Manager (D9–12) · Freelancer

| Paso | Acción |
|------|--------|
| 5.1 | Campaña(s) según tier |
| 5.2 | Ad groups audiencia + presupuesto |
| 5.3 | Subir creatividades + copy (texto overlay doc) |
| 5.4 | Revisión políticas TikTok pre-submit |

### Fase 6 — QA y launch (D12–14) · QA + Freelancer

Checklist §5 · QA G3 · activar · monitor 72h · informe learning.

### Fase 7 — Premium optimización (D15–45)

Rotación semanal creatividades · informes semana 2, 4, 30 · pausar fatigados · escalar winners.

## 6. Riesgos frecuentes

| Riesgo | Mitigación |
|--------|------------|
| Sin API — todo manual | Documentar en propuesta; no prometer dashboard NELVYON |
| Ad rechazado | Pre-check políticas; evitar before/after, claims médicos |
| Creative fatigue rápida | Rotación planificada; premium 8 videos |
| Landing no mobile | Bloquear launch; coordinar LANDING_SOP |
| Presupuesto insuficiente learning | Mínimo diario acordado en brief |
| Creator no entrega | Backup motion graphics en contrato |
