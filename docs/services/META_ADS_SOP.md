## Resumen

Estrategia, creatividades y lanzamiento de campañas Facebook/Instagram con pixel/CAPI verificado.

| Tier | Alcance | Plazo | Revisiones |
|------|---------|-------|------------|
| **Standard** | Setup 1 cuenta + hasta 2 campañas + 5 creatividades | 5–10 días laborables | 2 |
| **Premium** | Setup + video ads + 30d optimización | 15–20 días setup + 30d monitor | 4 |

## 9. Riesgos frecuentes

| Riesgo | Mitigación |
|--------|------------|
| Ad rechazado política | Review §special categories; pre-check copy |
| Pixel no dispara | Fase 3 obligatoria; Pixel Helper |
| Learning limitado sin budget | Mínimo presupuesto acordado en brief |
| Fatiga creativa | Rotación premium; 5+ ángulos iniciales |
| IG no vinculada | Verificar en Fase 0 |
| CAPI mal configurado | Doc eventos + test antes de escalar |

## 10. Flujo dentro de NELVYON OS

```
1. /os/clientes → cliente
2. /os/proyectos → "Meta Ads [Campaña]"
3. /os/tareas:
   T1 Auditoría BM | T2 Estrategia | T3 Tracking | T4 Creatividades | T5 Setup | T6 Launch | T7 Reporting
4. /os/entregables:
   - D1 Estrategia PDF
   - D2 Pack creatividades ZIP (internal hasta aprobación)
   - D3 Informes performance (client_visible)
5. Vincular en metadata proyecto: Ad Account ID, Pixel ID
6. QA: /os/ads-premium/preview (compartido Google/Meta)
7. Coordinar landing con proyecto LANDING OS si mismo cliente
```
