## 2. Proceso paso a paso

### Fase 0 — Intake (D0–1) · Account

| Paso | Acción | Evidencia |
|------|--------|-----------|
| 0.1 | Brief validado | `01-brief/` |
| 0.2 | OS: cliente + proyecto `LOGO-[CLIENTE]` | Creado |
| 0.3 | Kick-off 30 min | Acta |

### Fase 1 — Research (D1–2) · Freelancer

| Paso | Acción | Evidencia |
|------|--------|-----------|
| 1.1 | Benchmark 10 competidores (semiótica, color, tipografía) | Moodboard competencia |
| 1.2 | Definir territorio creativo (3 adjetivos diseño) | Nota 1 pág |
| 1.3 | Opcional: exploración IA conceptual (referencia, no final) | Carpeta refs |

### Fase 2 — Concepto (D2–4) · Freelancer

| Paso | Acción | Evidencia |
|------|--------|-----------|
| 2.1 | 2–3 concept boards según tier | PDF por dirección |
| 2.2 | Presentación rationale cada dirección | Loom 5 min |
| 2.3 | **Hito cliente:** elegir 1 dirección | Email aprobación |

### Fase 3 — Refinamiento (D4–6) · Freelancer

| Paso | Acción | Evidencia |
|------|--------|-----------|
| 3.1 | 2 variantes dentro dirección elegida | PDF |
| 3.2 | Ajuste tipografía/isotipo/color | Figma/AI |
| 3.3 | **Ronda revisión** según tier | Comentarios |
| 3.4 | Versión final aprobada para vectorizar | OK escrito |

### Fase 4 — Vectorización y kit (D6–7) · Freelancer

| Paso | Acción | Evidencia |
|------|--------|-----------|
| 4.1 | Master vector limpio (AI, SVG, PDF) | Archivos |
| 4.2 | Variantes: horizontal, vertical, isotipo, mono, invertido | Kit |
| 4.3 | Export PNG @1x/@2x/@4x, ICO, favicon | Carpeta exports |
| 4.4 | Mini brand sheet 1 pág | PDF |

### Fase 5 — Premium extras (D8–14) · Freelancer

| Paso | Acción |
|------|--------|
| 5.1 | Submarca o producto secundario |
| 5.2 | Animación logo 3–5s (MP4, Lottie si acordado) |

### Fase 6 — QA y entrega (D final) · QA + Account

| Paso | Acción |
|------|--------|
| 6.1 | Checklist §5 + QA G3 |
| 6.2 | OS entregable `client_visible` + handoff |

| ID | Criterio |
|----|----------|
| A1 | Logo legible a 24–32px (favicon test) |
| A2 | Funciona en fondo claro y oscuro |

| A5 | No depende de efectos para reconocimiento |
| A6 | Cliente aprobó dirección en Fase 2 por escrito |
| A7 | Kit completo según tier contratado |
| A8 | Premium: animación entregada en formatos acordados |

## 5. Checklist QA

- [ ] 🔴 Legible 32px favicon
- [ ] 🔴 Monocromo reconocible
- [ ] 🔴 SVG optimizado (SVGOMG o equivalente)
- [ ] 🔴 Zona de protección documentada
- [ ] 🟠 Usos incorrectos listados (brand sheet)
- [ ] 🟠 Licencia fuentes si tipografía custom outlined
- [ ] 🟡 Animación loop suave (premium)
- [ ] 🔴 QA G3 APROBADO
- [ ] 🔴 Entregable OS publicado

---

## 6. Riesgos frecuentes

| Riesgo | Mitigación |
|--------|------------|
| Cliente pide infinitas direcciones | Límite tier; change request |
| Logo raster entregado como “vector” | QA abre AI/SVG en Illustrator |
| Similaridad marca registrada | Disclaimer legal; búsqueda recomendada |
| Demasiados colores degradado | Máx. 3 colores regla SOP |
| IA generada sin vectorizar | Fase 4 obligatoria manual |

## 9. Perfil freelancer ideal

- **Rol:** Logo designer / identity designer
- **Experiencia:** 3+ años, 50+ marcas en portfolio
- **Skills:** Semiótica, tipografía, vector impecable, presentación conceptos
- **Entregable clave:** SVG production-ready sin trazado manual pendiente
- **Evitar:** Solo generadores IA sin refinamiento vector
